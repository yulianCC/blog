ez_unserialize

经典反序列化pop链

我们的目标是执行U类中的system()函数，从这里反推

  1. 最终执行: U::__invoke() -> N::__call(‘system’, …) -> system($_POST[‘cmd’])
    • 我们需要一个U类的实例被当作函数来调用，这样就会触发__invoke()。
  2. 触发 __invoke(): F::check()
    • F类中的check()方法会实例化一个由$this->finalstep指定的类，并将其作为函数调用：($this->step)()。
    • 我们可以将$this->finalstep设置为’u’。因为PHP的类名是大小写不敏感的，new ‘u’()会成功创建U类的实例。同时，使用小写的’u’可以绕过preg_match(“/U/”,$this->finalstep)的检查。
  3. 触发 check(): E::__get()
    • E类的__get()魔术方法会在我们尝试访问其私有属性secret时被调用。
    • 该方法会执行$this->found->check()。因此，我们将$this->found设置为我们构造好的F对象。
  4. 触发 __get(): V::__toString()
    • V类的__toString()方法会动态地访问一个属性：$this->go->$abc，其中$abc的值来自$this->dowhat。
    • 我们可以将$this->go设置为我们构造的E对象，并将$this->dowhat设置为字符串’secret’。这样就会触发对$e_object->secret的访问，进而调用E::__get()。
  5. 触发 __toString(): A::start()
    • A类的start()方法会执行echo $this->next。当echo一个对象时，会触发该对象的__toString()方法。
    • 因此，我们将$this->next设置为我们构造的V对象。
  6. 触发 start(): H::__destruct()
    • H类的__destruct()方法是我们的起点。当一个由unserialize()创建的对象在脚本结束时被销毁，这个方法就会被自动调用。
    • 该方法会执行$this->who->start()。我们将$this->who设置为我们构造的A对象。

整个利用链就构建完成：H::_destruct() -> A::start() -> V::_toString() -> E::_get() -> F::check() -> U::__invoke() -> system()

ida分析代码

是个自制的堆管理

这里存在uaf

函数在释放了堆块后，没有将存储指针的qword_5200[v2 + 512]清零 这意味着指针仍然指向之前被释放的内存区域

后续的edit或show操作如果使用这个索引，就会对一块已经free的内存进行读写，从而导致UAF漏洞

如前所述，由于 delete 函数不清空指针，攻击者可以先 delete 一个chunk，然后再次 edit 它。这允许攻击者向一块已经释放的内存中写入任意数据。

这对于篡改 tcache 或 fastbin 中 free chunk 的 fd 指针至关重要，从而可以实现任意地址分配

nbytes[0] <= (__int64)qword_5200[v2 + 528] 这个检查似乎可以防止溢出。但是，请注意 delete 函数会将 qword_5200[v2 + 528] 设置为0

如果攻击者 delete 一个chunk后再 edit 它，此时记录的大小为0。当用户输入一个大于0的size时，nbytes[0] <= 0 的检查会失败

但如果用户输入的size是0，read函数虽然不会读取数据，但检查会通过

如果这里存在有符号和无符号比较的整数漏洞，可能会绕过检查，但从伪代码看，这个检查是比较严谨的 主要问题还是与UAF配合使用

攻击者可以 delete 一个chunk，此时该chunk被放入 tcache 或 fastbin

这些空闲的chunk的头部会包含指向libc堆管理区域（如 main_arena）的指针

由于 delete 后指针不清零，攻击者可以调用 show 函数打印这个已经被 free 的chunk的内容

因为puts会一直打印直到遇到NULL字节，这就可以泄露出位于 main_arena 附近的libc地址，从而绕过ASLR

stack

p = malloc(0x1000) 分配 0x1000 字节（4096）

memset(p, 0, 0x2000)，从 p 开始写入 0x2000（8192）字节

直接越界写入 0x1000（4096）字节到 p 之后的内存

然后 buf = buf - 672：把 buf 指向分配块的前面 672 字节处 等于 p - 0x2A）

也就是说，后续所有以 buf 为基址的偏移都可以访问并覆盖 p 之前的数据

算 qword_4018 = buf + 256。注意 ((QWORD*)buf)[32] 的位置（32*8 = 256）恰好是 buf+256

程序把 ((QWORD)buf + 32) = buf + 4096（在该偏移写入 buf+4096）

接着又写 *(qword_4018 + 8) = sub_132E（也就是在 buf+264 写入函数指针 sub_132E）

程序预写了两处重要的指针到 buf 偏移 256/264

但是 recall 之前对 buf 的 memset 和后面 read(0, buf, 0x2000) 都会覆盖这些位置

攻击者可以把这些位置写成任意值或伪造结构

read(0, buf, 0x2000) 从 stdin 读入 0x2000 字节到 bu

fbuf 是 p - 672，read 会覆盖一大片内存，包括堆元、全局区、可能的 libc 指针表等

qword_4028 用于计数：如果 >2 则退出

最多 3 次触发 read 的机会

分析代码

代码实现了一个基于映射表的替换解密算法

输入是一个十六进制字符串 s，以空格为分隔符。

通过 s.split() 将其解析为一个十六进制字符串列表 lst。此列表构成了待解密的密文单元序列。脚本预定义了一个明文字符集 s (a-z, A-Z, 0-9, 等) 和一个字节序列 b。

通过迭代这两个序列，程序构建了一个字典 d，其键（key）是 b 中的字节值，值（value）是 s 中对应位置的字符。这个字典本质上是一个从加密后的字节到原始明文字符的逆向查找表（byte -> char）。

脚本遍历 lst 中的每一个十六进制字符串。

对每个元素，使用 int(hex_string, 16) 将其从十六进制表示转换为整数形式的字节值。

将此字节值作为键在字典 d 中进行查找，获取其映射的原始明文字符。

将所有检索到的字符顺序拼接，最终还原出完整的明文

def is_valid_idcard(s: str) -> bool:
    """校验身份证号码是否有效。"""
    cleaned_s = s.replace(" ", "").replace("-", "")
    if len(cleaned_s) != 18:
        return False
    if not cleaned_s[:17].isdigit() or not (cleaned_s[17].isdigit() or cleaned_s[17].upper() == 'X'):
        return False
    try:
        datetime.strptime(cleaned_s[6:14], '%Y%m%d')
    except ValueError:
        return False
    s_sum = sum(int(cleaned_s[i]) * ID_CARD_WEIGHTS[i] for i in range(17))
    expected_checksum = ID_CARD_CHECKSUM[s_sum % 11]
    return cleaned_s[17].upper() == expected_checksum

def is_valid_phone(s: str) -> bool:
    """校验手机号码是否有效。"""
    temp_s = s.strip()
    if temp_s.startswith("+86"):
        temp_s = temp_s[3:].strip()
    elif temp_s.startswith("(+86)"):
        temp_s = temp_s[5:].strip()
    cleaned_s = temp_s.replace(" ", "").replace("-", "")
    return len(cleaned_s) == 11 and cleaned_s.isdigit() and cleaned_s[:3] in PHONE_PREFIXES

def is_valid_bankcard(s: str) -> bool:
    """使用 Luhn 算法校验银行卡号是否有效。"""
    if not (16 <= len(s) <= 19 and s.isdigit()):
        return False
    digits = [int(d) for d in s]
    for i in range(len(digits) - 2, -1, -2):
        doubled = digits[i] * 2
        digits[i] = doubled - 9 if doubled > 9 else doubled
    return sum(digits) % 10 == 0

def is_valid_ip(s: str) -> bool:
    """校验IPv4地址是否有效。"""
    parts = s.split('.')
    if len(parts) != 4:
        return False
    for part in parts:
        if not part.isdigit() or (len(part) > 1 and part.startswith('0')) or not 0 <= int(part) <= 255:
            return False
    return True

    """校验MAC地址是否有效。"""

def extract_files_from_pcap(pcap_file, target_uri, output_dir):
    if not os.path.exists(output_dir):
        os.makedirs(output_dir)

    display_filter = f'http.request.method == "POST" && http.request.uri contains "{target_uri}"'
    
    cap = pyshark.FileCapture(pcap_file, display_filter=display_filter, keep_packets=False, tshark_path=r"D:\Wireshark\tshark.exe")

    count = 0
    
    for packet in cap:
        try:
            if not hasattr(packet, 'http'):
                continue

            http_layer = packet.http

            content_type = getattr(http_layer, 'content_type', '')
            if 'multipart/form-data' not in content_type:
                continue

            if not hasattr(http_layer, 'file_data'):
                continue
            
            raw_http_body = binascii.unhexlify(http_layer.file_data.replace(':', ''))

            headers = f'Content-Type: {content_type}\r\n\r\n'.encode('utf-8')
            msg_data = headers + raw_http_body

            msg = BytesParser(policy=default).parsebytes(msg_data)

            if msg.is_multipart():
                for part in msg.iter_parts():
                    filename = part.get_filename()
                    
                    if filename:
                        safe_filename = os.path.basename(filename)
                        
                        file_content = part.get_payload(decode=True)
                        
                        if file_content:
                            save_path = os.path.join(output_dir, safe_filename)
                            
                            base, extension = os.path.splitext(save_path)
                            counter = 1
                            while os.path.exists(save_path):
                                save_path = f"{base}_{counter}{extension}"
                                counter += 1

                            with open(save_path, 'wb') as f:
                                f.write(file_content)
                            
                            print(f"[{packet.number}] '{filename}' -> '{os.path.basename(save_path)}' ({len(file_content)} bytes)")
                            count += 1

        except Exception as e:
            print(f"[{packet.number}] '{filename}' -> An error occurred while processing")
            continue

    cap.close()
    print(f"Processed {count} streams")

if __name__ == '__main__':
    extract_files_from_pcap(PCAP_FILE, TARGET_URI, OUTPUT_DIR)

import easyocr
import os
import re
import pandas as pd
from tqdm import tqdm
import shutil

class IDCardExtractor:
    def __init__(self, images_dir='idcards/', fail_dir='fail/', use_gpu=True):
        print("Initializing EasyOCR (GPU)")
        try:
            self.reader = easyocr.Reader(['ch_sim', 'en'], gpu=use_gpu)
            if use_gpu:
                print("EasyOCR loaded")
        except Exception as e:
            print(f"Unable to initialize EasyOCR using GPU ({e})")
            print("Initializing EasyOCR (CPU)")
            self.reader = easyocr.Reader(['ch_sim', 'en'], gpu=False)
            print("EasyOCR loaded")
        
        self.images_dir = images_dir
        self.fail_dir = fail_dir
        
        os.makedirs(self.images_dir, exist_ok=True)
        os.makedirs(self.fail_dir, exist_ok=True)

    def extract_info_from_text(self, text):
        name = re.findall(r'姓名(.{2,4}?)性别', text)
        gender = re.findall(r'性别(.)民族', text)
        nation = re.findall(r'民族(.{1,3}?)出生', text)
        address = re.findall(r'住址(.*?)公民身份号码', text)
        number = re.findall(r'公民身份号码([0-9Xx]{18})', text)

        data = {
            '姓名': name[0] if name else '未识别',
            '性别': gender[0] if gender else '未识别',
            '民族': nation[0] if nation else '未识别',
            '地址': address[0].strip() if address else '未识别',
            '身份证号码': number[0] if number else '未识别'
        }

        return data

    def process_and_save_incrementally(self, output_path='result.csv'):
        image_files = [f for f in os.listdir(self.images_dir) if f.lower().endswith(('png', 'jpg', 'jpeg', 'bmp'))]

        write_header = not os.path.exists(output_path)

        print(f"len(image_files) files")
        
        total_processed = 0
        fail_count = 0
        for image_name in tqdm(image_files):
            image_path = os.path.join(self.images_dir, image_name)
            
            try:
                ocr_result = self.reader.readtext(image_path, detail=0)
                full_text = "".join(ocr_result).replace(" ", "")
                extracted_data = self.extract_info_from_text(full_text)

                if '未识别' in extracted_data.values():
                    tqdm.write(f"Unable to OCR '{image_name}'")
                    dest_path = os.path.join(self.fail_dir, image_name)
                    shutil.copy(image_path, dest_path)
                    tqdm.write(f"    -> Copied to '{self.fail_dir}'")
                    fail_count += 1
                    continue
                
                df_single_row = pd.DataFrame([extracted_data])

                df_single_row.to_csv(
                    output_path, 
                    mode='a',
                    header=write_header,
                    index=False, 
                    encoding='utf-8-sig'
                )

                write_header = False
                total_processed += 1

            except Exception as e:
                tqdm.write(f"An error occurred ({e})")
                
                dest_path = os.path.join(self.fail_dir, image_name)
                shutil.copy(image_path, dest_path)
                tqdm.write(f"    -> Copied to '{self.fail_dir}'")
                fail_count += 1

                error_data = {
                    '姓名': f'处理失败: {image_name}',
                    '性别': 'N/A', '民族': 'N/A', '地址': 'N/A', '身份证号码': 'N/A'
                }
                df_error_row = pd.DataFrame([error_data])
                df_error_row.to_csv(
                    output_path, 
                    mode='a', 
                    header=write_header, 
                    index=False, 
                    encoding='utf-8-sig'
                )
                write_header = False

        tqdm.write(f"Processed {total_processed} files")
        if fail_count > 0:
            tqdm.write(f"{fail_count} unexpected files copied to '{self.fail_dir}'")


if __name__ == '__main__':
    extractor = IDCardExtractor(images_dir='idcards/', fail_dir='fail/', use_gpu=True)
    
    output_file = 'result.csv'
    
    extractor.process_and_save_incrementally(output_path=output_file)

def is_valid_idcard(s: str) -> bool:
    cleaned_s = s.replace(" ", "").replace("-", "").upper()
    if len(cleaned_s) != 18: return False
    if not cleaned_s[:17].isdigit() or not (cleaned_s[17].isdigit() or cleaned_s[17] == 'X'): return False
    try: datetime.strptime(cleaned_s[6:14], '%Y%m%d')
    except ValueError: return False
    s_sum = sum(int(cleaned_s[i]) * ID_CARD_WEIGHTS[i] for i in range(17))
    expected_checksum = ID_CARD_CHECKSUM[s_sum % 11]
    return cleaned_s[17] == expected_checksum

def is_valid_phone(s: str) -> bool:
    temp_s = s.strip()
    if temp_s.startswith("+86"): temp_s = temp_s[3:].strip()
    elif temp_s.startswith("(+86)"): temp_s = temp_s[5:].strip()
    cleaned_s = temp_s.replace(" ", "").replace("-", "")
    return len(cleaned_s) == 11 and cleaned_s.isdigit() and cleaned_s[:3] in PHONE_PREFIXES

def parse_http_form_data(form_data_str):
    try:
        parsed_data = parse_qs(form_data_str)
        return {k: unquote(v[0]) for k, v in parsed_data.items()}
    except Exception:
        return {}

def process_pcap_performant(filepath, tshark_path):
    display_filter = 'http.request.method == "POST" && (http.request.uri == "/login.php" || http.request.uri == "/survey.php")'
    print(f"Fliter: {display_filter}")
    
    streams = {}
    
    try:
        count_cmd = [
            tshark_path, '-r', filepath, '-Y', display_filter, '-T', 'fields', '-e', 'frame.number'
        ]
        proc = subprocess.run(count_cmd, capture_output=True, text=True, check=True)
        total_packets = len([line for line in proc.stdout.splitlines() if line])
        print(f"Total {total_packets} packages")

        cap = pyshark.FileCapture(
            filepath,
            tshark_path=tshark_path,
            display_filter=display_filter,
            keep_packets=False
        )
        
        for packet in tqdm(cap, total=total_packets):
            try:
                stream_index = int(packet.tcp.stream)
                if stream_index not in streams:
                    streams[stream_index] = {}
                
                uri = packet.http.request_uri

                form_data_str = None
                if hasattr(packet.http, 'urlencoded_form'):
                    form_data_str = packet.http.urlencoded_form.value
                elif hasattr(packet.http, 'file_data'):
                    raw_body = binascii.unhexlify(packet.http.file_data.replace(':', ''))
                    form_data_str = raw_body.decode('utf-8', errors='ignore')

                if not form_data_str:
                    continue

                form_data = parse_http_form_data(form_data_str)

                if uri == '/login.php':
                    streams[stream_index].update(form_data)
                elif uri == '/survey.php':
                    if 'gender' in form_data: form_data['sex'] = form_data.pop('gender')
                    if 'birth_date' in form_data: form_data['birth'] = form_data.pop('birth_date').replace('/', '')
                    streams[stream_index].update(form_data)

            except Exception:
                continue

    extracted_data = [data for data in streams.values() if 'username' in data and 'name' in data]
    print(f"Total {len(extracted_data)} HTTP sessions")
    return extracted_data


def validate_and_clean_data(records, id_card_map):
    print("Cleaning")
    valid_records = []
    
    for record in tqdm(records):
        required_keys = ['username', 'password', 'name', 'sex', 'birth', 'phone']
        if not all(key in record for key in required_keys): continue
        name = record.get('name')
        if name in id_card_map:
            record['idcard'] = str(id_card_map[name])
        else:
            print(f"ID corresponding to the name '{name}' cannot be found, skip")
            continue
        
        phone, idcard = record['phone'], record['idcard']
        if not is_valid_phone(phone): continue
        if not is_valid_idcard(idcard): continue
        
        birth_from_idcard = idcard[6:14]
        if record['birth'] != birth_from_idcard: continue
            
        sex_digit = int(idcard[16])
        sex_from_idcard = '男' if sex_digit % 2 != 0 else '女'
        if record['sex'] != sex_from_idcard: continue

        valid_records.append(record)
        
    print(f"Cleaned. Total {len(valid_records)} valid data")
    return valid_records

def write_to_csv(records, filepath):
    fieldnames = ['username', 'password', 'name', 'sex', 'birth', 'idcard', 'phone']
    with open(filepath, 'w', newline='', encoding='utf-8') as csvfile:
        writer = csv.DictWriter(csvfile, fieldnames=fieldnames)
        writer.writeheader()
        for record in records:
            writer.writerow({key: record.get(key) for key in fieldnames})

再次查看 “flag生成器.exe” 对 Key 的校验逻辑

即输入的值的前 32 位应该是该预定值的加密

def encrypt(text):
    key = [3, 5, 2]
    key_len = len(key)

    encrypted = []

    for i, ch in enumerate(text):
        if ch.isalpha():
            base = ord('a') if ch.islower() else ord('A')
            shift = key[i % key_len]
            enc_ch = chr((ord(ch) - base + shift) % 26 + base)
            encrypted.append(enc_ch)
        else:
            encrypted.append(ch)

    return ''.join(encrypted)
